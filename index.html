Resources for New Production Editors

Commonly Used Terms

Here are some of the commonly used terms/names at O'Reilly:

FTP Date
Files to Printer Date. The date that you send the completed book to the printer. The "last FTP date" is the last date of the month that you can send a book to the printer and have it released that same month.

Printer Info

Most O'Reilly books are printed at LSI. Some books requiring more advanced color printing go elsewhere. Once a book is ready, the production editor sends the interior PDF and the cover PDF to the printer. These links detail how to send books to the different printers.

Sending LSI Files to CoreSource: this is how to send books to LSI

Printer Codes: a list of the various printer codes that go on the copyright pages. It also notes when a printer is located outside the United States

Miscellaneous

PitStop Profiles: a list of commonly used PitStop profiles. PitStop profiles are scripts you can run on a PDF using the command line that will change the color or alter the trim size. This is usually done in order to meet printer specifications. Note that you must be on the Touchstone server to use these profiles
